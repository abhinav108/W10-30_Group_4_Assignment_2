Cyber Security



Collection of computer networks interconnected with each other, generally termed as the Internet has revolutionized globally and have made a big impact on human lives. The Internet is decentralized which means that anyone can contribute and share information and so applications like websites and apps are hosted publically on servers. Attacks on these publically accessible servers and computer networks could prove disastrous outcomes.

The term is used in a variety of contexts, ranging from business to mobile computing, and it can be broken down into a few categories:

Network security

It is the security guidelines and practices to prevent and reveal unauthorized access, misuse, modification, or denial of a computer network and network-accessible resources. There are two types of network threats, namely, Passive Network Threats and Active Network Threats. Passive threats in Information security are an attempt to gain information about the target without changing the data whereas Active threats in Information security are an attempt to gain unauthorized access to the computer network either to alter system resources or affect the performance of the computer system (Kaspersky 2021). Denial of service (DoS) attack is considered as one of the most concerning cyber threats of all time.

Application security

It increases the security of apps by detecting and correcting code flaws. If the application is hacked, it may allow access to data that was supposed to be safe and secure.

Information security or data security

It protects data from unauthorised access or modifications both when it is stored and when it is transferred from one machine to another.

Operational security (OPSEC)

In this process companies gain and secure people’s information regarding themselves.It is an analyzing process where data assets analyzed and find out what is necessary to control these data assets

There are different tricks used by cyber criminals to target their victims or in other words there different types of cyber security threats

Malware- It is a malicious software created and transferred by criminals to the target's system. It is one of the most common cyber threats.

Trojans- It is similar to malware which is tricked by hackers. Initially these Trojans look like simple software but after some time they cause damage to the system's data.

Phishing- It is a process to target victims sensitive information like user id and password through email by creating a fake login page.
